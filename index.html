Ervaring

Voor mijn MBO opleiding heb ik een paar maanden stage gelopen.

Voor deze stage ben ik ook twee keer in het buitenland geweest.
Daarnaast ben ik ook bezig geweest voor mijzelf, vooral om beter te leren designen.

Mijn portfolio is te vinden op mau.bio.
